Project: Readito

Description:
Offline-first desktop reader for EPUB, PDF and image folders.
Built with Tauri 2 (Rust) + React + Vite + TypeScript.
MIT licensed.

Core principles:
- online-first
- Local file ownership
- Lightweight

Release writing constraints:
- No emojis
- No exaggerated marketing tone
- No repetition of unchanged features
- No invented functionality
- Only describe changes present in git history
